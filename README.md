# 冲了吗

温暖克制的个人每日打卡工具。核心只有“没冲”和“冲了”两个选择，支持本地持久化、历史修改、月份切换、连续天数、月度分布、横向日历，以及可展开的「蛋蛋恢复仓」时间趋势可视化。

排行榜包含「忍者榜」与「连冲榜」：前者按当前连续没冲天数排序，后者按当前连续冲了天数排序。用户可设置 3—16 位自定义 ID，并随时选择公开参与或不公开。选择公开时，只同步自定义 ID 和两项连续天数；选择不公开时，会从共享排行榜删除资料，ID 与完整打卡记录只留在本机。

「蛋蛋恢复仓」会读取最近一次“冲了”的日期，也支持手动修改到精确时间。恢复百分比仅依据时间曲线估算，120 小时后封顶，不代表真实精液量、精子数量、身体检测结果或医学诊断。

详细打卡记录始终只保存在当前设备浏览器中。网站可安装为 PWA，并可由用户主动开启固定时间的每日推送；提醒服务只保存匿名推送地址、提醒时间与时区，不上传当天打卡状态。共享排行榜采用荣誉制，并已连接公开的持久化接口；如需迁移或自建服务，参见 [`leaderboard-worker/README.md`](leaderboard-worker/README.md)。
